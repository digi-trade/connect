# 简介

欢迎使用Cabital FaaS API，FaaS API 使合作伙伴能够合规、合法的将Cabital的法币入金途径集成到合作伙伴的系统中。

API 功能简介：

- 提供用户侧的 Authentication 与 Authorization，打通双方的用户体系
- 提供用户的账号余额，以及划转到合作伙伴方的同名账户，并支持C+T（转换+划转）
- 通过API的相互调用，确认用户的同名，以及交换KYC基础信息，支持合规的要求
